Overview:

xxxTerminal.com was developed as a Platform as a Service (PaaS) solution aimed at cryptocurrency traders.
The platform provided users with the infrastructure and tools necessary to deploy, manage, and monitor automated crypto trading strategies in a secure, cloud-based environment. Although the project was active only during 2022, it involved building a comprehensive system from infrastructure deployment to user-facing features.

Key Features:

  • Cloud Infrastructure Deployment
    • Successfully deployed and managed a scalable LAMP (Linux, Apache, MySQL, PHP) stack on Amazon Web Services (AWS).
    • Utilized core AWS services including EC2 for compute instances, S3 for reliable object storage (e.g., logs, user data backups), and Route53 for DNS management and routing.
  • Security Implementation
    • Implemented Short-term Access Tokens minimizing the damage if credentials leaks.
    • Integrated Two-Factor Authentication (2FA) as a mandatory layer for critical actions.
    • Established Email verification process with self-hosted SMTP server for enhanced security & controls.
  • Content Management & Internationalization
    • Developed a custom Content Management System (CMS) module to manage platform content (e.g., articles, guides, announcements).
    • Integrated the Google Cloud Translation API to provide automatic language translation, making the platform accessible to a global audience.
  • User Acquisition & Growth
    • Designed and implemented a Referral Program (Affiliate System) to incentivize user growth.
    • The program offered commission-based rewards to users who successfully referred new paying subscribers, fostering organic platform adoption.

Technical Stack

LAMP Stack:

  • Backend : PHP, Codeignitor framework
  • Frontend : Bootstrap, jQuery
  • Database : MariaDB
  • Cloud Services : AWS (EC2, S3, Route53)
  • Web Server : Apache Web Server
