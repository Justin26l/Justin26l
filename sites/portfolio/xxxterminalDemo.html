Overview:

xxxTerminal.com provide algo trading solution aimed at cryptocurrency traders.
The platform provide a graphical interface to manage a trading bots, make traders to deal with software & infrastructure in ease.

Key Features:

  • Cloud Infrastructure Deployment
    • Deployed LAMP stack (Linux, Apache, MySQL, PHP) on Amazon Web Services.
    • Utilized AWS services including :
      • EC2 for PHP compute & Mysql database instances
      • S3 for store images & DB backups
      • Cloudfront for CDN network.
  • Security Implementation
    • Short-lived tokens to minimizing the damage if credentials leaks.
    • Time-based One-time-password as Two-Factor Authentication to secure critical actions.
    • Email security notice with self-hosted SMTP server for enhanced security.
  • Content Management & Internationalization
    • Content Management System (CMS) module developed to manage platform content (e.g., articles, faq, resources).
    • Automatic language translation achieve through Google Cloud Translation API, make contents accessible to global audience.
  • User Acquisition & Growth
    • Referral / Affiliate Program to incentivize user growth.
    • Offer Commission-based rewards to who bring new subscribers.
  • Trading Bot Kernel
    • Writen in TypeScript, running on NodeJs, with multi threading (worker threads) design
    • handle data stream from exchange's web socket

Technology Stack

LAMP Stack:

  • Backend : PHP, Codeignitor
  • Frontend : Bootstrap, jQuery
  • Database : MariaDB
  • Web Server : Apache
  • Cloud Services : AWS (EC2, S3, Route53, Cloudfront)
